Overview:
This project introduces an innovative approach to controlling PowerPoint presentations using hand gestures, eliminating the need for traditional input devices such as a keyboard or mouse. By leveraging computer vision and gesture recognition techniques, the system enhances user experience by enabling touchless navigation through slides.

Technologies Used:

OpenCV (Computer Vision)
Python
MediaPipe (Hand Tracking)
PyAutoGUI (Automating PPT Controls)
Key Features:

Slide Navigation (Next/Previous)
Start and Stop Presentation Mode
Gesture-based Interaction for a Seamless Experience
User-friendly and Efficient
Applications:

Touchless Presentation Control for Educators & Professionals
Enhanced Accessibility for Individuals with Disabilities
Smart Conference Room Automation
Conclusion:
This project demonstrates the practical implementation of hand gesture recognition in real-world applications, improving the ease of interacting with presentation software. The system offers a natural and intuitive way to control slides, making presentations more dynamic and engaging.
